# Investing Tesis & Cursos - Landing Page

Landing page comercial para **Investing Tesis & Cursos**.

Slogan: **Invierte hoy en tu futuro académico.**

## Tecnologías

- React
- Vite
- Tailwind CSS
- Framer Motion
- Lucide React

## Ejecutar localmente

```bash
npm install
npm run dev
```

Luego abre la URL local que indique Vite.

## Compilar para producción

```bash
npm run build
```

## Publicación recomendada

Se recomienda publicar en Vercel o Netlify por facilidad de despliegue continuo desde GitHub.
